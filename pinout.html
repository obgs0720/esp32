市售元件教學：LED 燈光控制

本頁面將介紹如何使用 ESP32 的數位輸出功能，控制外部發光二極體（LED）的亮滅與基本閃爍邏輯。

1. 所需材料清單

  • ESP32 開發板 x 1
  • 5mm LED 燈（顏色不限） x 1
  • 220 歐姆電阻 x 1 (保護 LED 避免燒毀)
  • 麵包板 x 1
  • 杜邦線（公對公） x 數條

2. 腳位與接線說明

LED 具有正負極之分：長腳為正極（Anode），短腳（或瓶身平切面處）為負極（Cathode）。

LED 元件端 連接對象 ESP32 開發板端
長腳 (正極) 接 220Ω 電阻的一端，電阻另一端接... GPIO 23
短腳 (負極) 直接連接 GND (接地)

4. 範例程式碼 (Arduino IDE)

請將以下程式碼複製到您的 Arduino IDE 中，並選擇正確的 ESP32 開發板進行燒錄。

// 定義控制 LED 的 GPIO 腳位
const int ledPin = 23;

void setup() {
  // 將 GPIO 23 設定為輸出模式
  pinMode(ledPin, OUTPUT);
}

void loop() {
  digitalWrite(ledPin, HIGH); // 輸出高電位，點亮 LED
  delay(1000);                // 等待 1 秒鐘 (1000 毫秒)
  
  digitalWrite(ledPin, LOW);  // 輸出低電位，熄滅 LED
  delay(1000);                // 再等待 1 秒鐘
}

5. 常見問題與除錯

💡 為什麼我的 LED 不會亮？
1. 檢查 LED 正負極是否接反（長腳必須透過電阻接在 GPIO，短腳接 GND）。
2. 確認程式碼中的 ledPin 數字與您實體接線的 GPIO 號碼完全一致（注意：是 GPIO 號碼，而非開發板上的引腳順序編號）。
3. 確認電阻阻值是否過大（如誤用 10K 歐姆電阻會導致電流過小使 LED 不亮）。
